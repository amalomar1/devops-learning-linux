## 1. touch
Creates a new empty file. For example, `touch hello.txt` creates an empty file called hello.txt.

## 2. grep
Searches for specific text inside a file. For example, `grep "error" hello.txt` looks for the word "error" inside hello.txt.

## 3. pwd
Displays the present working directory (the folder I am currently in).

## 4. echo + >>
`echo` prints text to the terminal. For example, `echo "hello there"` prints text.
The `>>` symbol appends text to a file. For example, `echo "how are you?" >> hello.txt` adds that text to the end of hello.txt.

## 5. whoami
Shows the current logged-in username.
