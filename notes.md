## File System Navigation Practice

I practiced basic Linux navigation and file commands.

## Commands I used and what they do:

- cd /var/log → Changes directory to system log folder
- ls -lah → Lists all files with details like size, permissions, hidden files
- pwd → Shows current directory path
- touch test.txt → Creates a new empty file
- mkdir -p projects/demo → Creates folders (including parent folders if needed)
- cp test.txt projects/demo/ → Copies a file into another directory
- mv projects/demo/test.txt projects/demo/backup.txt → Renames a file
- rm projects/demo/backup.txt → Deletes a file
- cat /etc/passwd → Displays system user accounts
- head -n 20 /etc/services → Shows first 20 lines of a file
- tail -f /var/log/auth.log → Shows live log updates
